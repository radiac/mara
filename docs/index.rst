=======================================
Mara - Python network service framework
=======================================

Mara is an asynchronous event-based python framework designed for building TCP/IP
services, such as telnet, HTTP and websocket servers.

* Project site: https://radiac.net/projects/mara/
* Source code: https://github.com/radiac/mara
